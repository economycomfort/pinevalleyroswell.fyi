# How to Post News & Updates

## Quick Start

1. Go to the `content/news/` folder in the GitHub repository
2. Click **Add file → Create new file**
3. Name the file: `YYYY-MM-DD-short-title.md` (e.g., `2026-04-01-spring-cleanup.md`)
4. Copy the contents of `POST_TEMPLATE.md` into the new file
5. Fill in the title, date, description, and body
6. Click **Commit changes** (add a brief note like "Add spring cleanup post")
7. The site will automatically update within about 60 seconds

## File Name Format

`YYYY-MM-DD-your-post-title.md`

- Use the date of the post
- Use lowercase letters and hyphens, no spaces
- Examples:
  - `2026-06-15-annual-meeting.md`
  - `2026-07-04-independence-day-cookout.md`

## Tips

- The `description` field (in the header) is the preview text shown on the News index page — keep it to one sentence
- You don't need to know HTML — plain text and the simple formatting shown in POST_TEMPLATE.md is all you need
- If you make a mistake, you can edit the file directly in GitHub and commit the correction
