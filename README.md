# Pine Valley HOA Website

Static site built with [Hugo](https://gohugo.io) and hosted on Cloudflare Pages.

## Local Development

```bash
hugo server -D
```

Visit http://localhost:1313

## Adding Content

See [AUTHORING.md](./AUTHORING.md) for step-by-step instructions on posting news and updates.

## Deployment

Pushes to `main` automatically trigger a Cloudflare Pages build and deploy.
